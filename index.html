Research projects

Confluo (ongoing)
Confluo is a system for real-time monitoring and analysis of data, that supports high-throughput concurrent writes of millions of data points from multiple data streams; online queries at millisecond timescale; and ad-hoc queries using minimal CPU resources.

Succinct
Succinct is a data store that enables efficient queries directly on a compressed representation of the input data.
