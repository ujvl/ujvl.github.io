Parallel Computing Application Example

To be able to compare instrumental observations with a baseline sky catalog of the development of the universe, an N-body simulation framework called HACC (Hardware/Hybrid Accelerated Cosmology Code) was developed. It was designed to run efficiently on several high performance computing architectures and to scale to millions of cores. It uses a Particle-Mesh solver and MPI for long-range forces, and depending on node architecture, switches between a tree and particle-particle algorithm for short-range forces.
